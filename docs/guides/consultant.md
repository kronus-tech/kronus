# Using Kronus for Consulting

Kronus writes proposals, tracks clients, organizes project knowledge, and helps you deliver faster.

## 5 Things to Try

### 1. Write a proposal
> "Write a proposal for [project]. Budget around [amount]. Timeline: [weeks]. The client needs [outcomes]."

Kronus drafts a professional proposal with scope, timeline, pricing, and deliverables. It learns from past proposals over time.

### 2. Client intake
> "Create a client intake questionnaire for [type of engagement]."

Generates a structured questionnaire you can send to new clients, saved as a reusable template.

### 3. Status reports
> "Create a weekly status report template for [project]. Include: accomplishments, next steps, risks, and decisions needed."

Saved as a note — ask Kronus to "fill in this week's status report" each week.

### 4. Lessons learned
> "Organize my notes from the [project] engagement into lessons learned and reusable knowledge."

Kronus extracts insights from your project notes and saves them for future reference.

### 5. Competitive analysis
> "Compare [option A] vs [option B] for [client situation]. Consider cost, timeline, risk, and long-term value."

Structured analysis you can include in recommendations or proposals.

## Your Knowledge Base

- **Clients/** — client information and engagement tracking
- **Projects/** — active project work
- **Proposals/** — proposals and SOWs
- **Knowledge-Base/** — reusable insights and templates
- **Archive/** — completed engagements
